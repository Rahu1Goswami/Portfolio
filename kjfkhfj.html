Hotel Management System

  • Developed a Comprehensive Hotel Management System with real-time room status updates.
  • Receptionists can view live availability of rooms and allot rooms based on customer requirements.
  • Room statuses (available, booked, under maintenance, etc.) are automatically updated in the database.
  • Used HTML, CSS and JS for web app developement and MYSql for backend database.
  • Learn More

Home Page, Gives entire transaction History at a glance. New bookings can be made through 'Add Booking' button.

Employee Management Page, Admin can add new Employee or access details of existing ones.

After clicking on 'Add Booking' or 'Add Employee' a new page asks for details before verifying it and storing it in the database.

Room Management Page, shows current status of each room. Status is automatically updated according to check out dates and new bookings.

Guest Details, Details about all the guests till last Database update can be fetched through this page.
